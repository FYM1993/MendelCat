# 《孟德尔的小猫王朝》项目说明书

## 1. 游戏核心愿景 (Core Vision)

《孟德尔的小猫王朝》是一款融合了 **遗传学模拟**、**王朝叙事** 与 **策略养成** 的轻量化 2D 游戏。玩家通过管理家族血脉，在「视觉美学（稀有基因）」与「国家强度（六维属性）」之间寻找动态平衡，谱写一部跨越百年的猫咪家族史。

---

## 2. 核心玩法机制 (Core Gameplay)

### 🧬 A. 孟德尔遗传系统 (Genetic System)

游戏底层构建了完整的双倍体遗传模型：

- **基因对 (Gene Pair)**：每只猫咪拥有 `[父本, 母本]` 组成的等位基因。
- **显隐性逻辑**：遵循孟德尔第一定律。显性基因 (Dominant) 优先表现，隐性基因仅在纯合状态下表达。
- **共显性 (Codominance)**：特定显性基因组合（如 Flame + Ice）将触发性状融合，生成全新的合成性状（如 Steam）。
- **稀有突变**：每次繁育有 1% 概率跳出遗传限制，觉醒「神话血统 (Mythic)」。

### 📊 B. 六维属性权重 (Six-Dimensional Stats)

猫咪属性影响游戏各层级的数值检定：

| 属性 | 作用 |
|------|------|
| **体质 (CON)** | 决定自然寿命上限与王朝稳定性 |
| **力量 (STR)** | 决定武力夺权的胜率与外部扩张门槛 |
| **敏捷 (AGI)** | 决定规避刺杀的成功率及资源获取效率 |
| **智力 (INT)** | 决定政治阴谋成败与盟约加成效率 |
| **魅力 (CHA)** | 决定联姻成功率及流浪猫（基因库）造访概率 |
| **幸运 (LUK)** | 影响基因突变率与探险掉落 |

### 👑 C. 王朝更迭与血统决策 (Succession & Bloodline)

玩家在每一代更迭时面临战略抉择：

- **邻国王室 (Royal)**：数值稳健，提供强大的「国家盟约」Buff，用于维持属性天花板。
- **荒野流浪猫 (Wild)**：极端的「偏科天才」，携带稀有外观基因，但伴随严重的体质缺陷（短命风险）。
- **权力博弈**：每一胎后代中属性最强者并不一定能稳坐王位，兄弟姐妹间会根据智力/力量差值触发夺权事件。

### 🗺️ D. 地缘政治盟约 (Kingdom Pacts)

世界由五个国家组成，每个国家拥有独特的偏好属性。通过联姻获得的盟约会根据「母系血缘」动态激活，为当届猫王提供全局属性倍率加成。
